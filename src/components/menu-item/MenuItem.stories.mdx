## Anatomy

<img src={menuItemAnatomyImg} alt={'menu item anatomy'} />

The Menu Item Component is made up of 4 key elements:

-   Label
-   Supporting Visual
-   Secondary Label
-   Metadata

### Label

This is the only required element of the Menu item. The label describes the action or selection and should be as concise
as possible.

### Leading Element (optional)

The leading Element (supporting visual) is used to offer quicker recognition of the menu item. Types of visuals here
could be an icon, avatar image, or an emoji. This is an optional element. See Variants below for visual references.
